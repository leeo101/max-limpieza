## 🎨 Personalización Fácil

### Cambiar WhatsApp
Archivo: `.env.local`
```env
NEXT_PUBLIC_WHATSAPP_NUMBER=5491123456789
```

### Cambiar Colores
Archivo: `app/globals.css`
```css
:root {
  --primary: #0ea5e9;
  --secondary: #10b981;
  --accent: #f59e0b;
}
```

### Agregar Productos
Desde: `/admin/productos` → "Agregar Producto"
